#Conway's Game of Life (Terminal ANSI Version)
Простая реализация игры «Жизнь» Конвея на C с цветной визуализацией в терминале.
# Описание
Программа моделирует клеточный автомат Game of Life с бесконечным циклом обновления.
Каждая клетка может быть живой или мёртвой, а также имеет «возраст», который влияет на её цвет.
#Особенности:
- Торроидальное поле (границы зациклены)- 
- Цветная визуализация через ANSI escape-коды
- Возраст клеток влияет на цвет
- Случайная генерация начального состояния
- Быстрая анимация в терминале
# Правила игры
Для каждой клетки:
- Живая клетка выживает при 2 или 3 соседях
- Умирает при <2 или >3 соседях
- Мёртвая клетка оживает при ровно 3 соседях
- Возраст клетки увеличивается при выживании
# Цвета
Цвет клетки зависит от её возраста:
- Молодые клетки — зелёные оттенки
- Старые клетки — переходят к красному
- Очень старые — ярко-красные
# Требования
- Компилятор C (gcc / clang)
- Терминал с поддержкой ANSI escape-кодов
(Linux, macOS, или Windows Terminal / PowerShell)
#Сборка
'''
gcc -O2 life.c -o life
'''
#Запуск
'''
./life
'''
#Управление
Программа работает бесконечно
Для выхода используйте:
'''
Ctrl + C
'''
